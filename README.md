# QA Office

Proyecto web para generar reportes instantáneos sobre las métricas del
departamento de QA para los proyectos de la empresa.

## Stack

- [Next.js](https://nextjs.org) (App Router, TypeScript)
- [Tailwind CSS](https://tailwindcss.com)
- Desplegado en [Vercel](https://vercel.com)

## Desarrollo local

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000).

## Despliegue

Cada push a `main` en GitHub genera un despliegue en Vercel.
